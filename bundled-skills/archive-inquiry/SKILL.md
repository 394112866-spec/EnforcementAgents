# 查档指引 Skill（执行查档"怎么去"指南）

## 功能概述

执行律师查档的最大痛点不是"查什么"——你已经从案件分析里知道了——而是**"去哪里查、带什么材料、几点开门、能不能提前预约、临场有什么坑"**。这些信息分散在小红书律师的随口分享、百度地图的评论、政府网站的角落公告里。

本 skill 做一件事：根据被执行企业注册地址，自动搜索小红书 MCP 和互联网公开信息，把零散的经验碎片拼接成一份**可以直接拿着出发的查档指引**。

**你去查档，不是去探索——是去执行一个已知坐标。**

## 使用场景

- 拿到被执行人企业名称，需要调取工商内档查实缴/股权转让
- 发现被执行人名下不动产，需要去不动产登记中心查册
- 追加被执行人配偶，需要调取婚姻登记信息
- 到一个不熟悉的城市办案，需要快速了解当地查档流程
- 当事人问"查这个要多久/多少钱"，需要给出准确答复

## 输入

**方式一：从 execution-survey 读取（推荐）**
```
根据财产调查报告，生成查档指引
```
自动读取 `.execution-survey.json` 中的企业名称 + 注册地址。

**方式二：手动指定**
```
生成查档指引：
企业：深圳市华宇元典信息服务有限公司
注册地址：广东省深圳市南山区科技园XX路XX号
查档类型：工商内档
```

查档类型选项：
- `工商内档`（默认） — 企业注册登记全套档案
- `不动产档案` — 房产登记信息查册
- `婚姻登记` — 被执行人婚姻登记信息（追加配偶时需要）

## 工作流程

### 第一步：确定查档坐标

**1.1 获取企业注册地址**

从 `.execution-survey.json` 的 `enterprise.address` 字段读取。

**1.2 解析城市和区县**

```
输入：广东省深圳市南山区科技园XX路XX号
解析：
  - 省份：广东省
  - 城市：深圳市
  - 区县：南山区
  - 查档类型：工商内档 → 深圳市市场监督管理局（南山分局/市民中心窗口）
```

**1.3 确认查档单位**

| 企业类型 | 查档单位 | 级别 |
|---------|---------|------|
| 公司（有限责任公司/股份公司） | XX市市场监督管理局 | 市级或区级 |
| 个体工商户 | XX区市场监督管理局 | 区级 |
| 合伙企业 | XX市市场监督管理局 | 市级 |
| 外资企业 | XX市市场监督管理局（外资窗口） | 市级 |

> **提示**：部分城市市局和区局都可以查，但内档完整度不同 — 市局最全。如不确定，优先列出市局。

--- 输出阶段提示 ---
```
📍 查档坐标确认：
- 企业：XX有限公司
- 注册地：广东省深圳市南山区
- 查档类型：工商内档
- 预计查档单位：深圳市市场监督管理局（企业档案查询窗口）

开始查询当地查档指引？（是/修改查档类型）
```

### 第二步：搜索当地查档信息

**本步骤是核心——调用小红书 MCP 和联网搜索，从不同渠道收集查档信息。**

#### 2.1 小红书 MCP 搜索（律师经验帖）

**搜索策略：三组关键词，分批搜索**

**第一组：查档经验帖**
```
搜索词示例：
- "[城市名] 查工商内档"
- "[城市名] 市场监督管理局 查档"
- "[城市名] 律师查档 企业档案"
- "[城市名] 执行 调企业内档"
```

提取信息：
- 窗口地址（具体到楼层窗口号）
- 实际工作时间（和官方公告可能不同）
- 现场流程（要不要取号/填什么表/等多久）
- 被拒的经历和解决方法
- 窗口工作人员的特殊要求
- 预约渠道（公众号/APP/网站）
- 打印费用
- 最近的停车场/地铁出口

**第二组：坑点总结帖**
```
搜索词示例：
- "[城市名] 查档 踩坑"
- "[城市名] 工商 查档 被拒"
- "[城市名] 查档 经验"
- "查工商内档 材料不全"
```

提取信息：
- 常见的材料被拒原因
- 哪些窗口不办查档（去错楼层/窗口）
- 介绍信抬头写错导致的拒办
- 临时通知不上班（培训/系统升级）

**第三组：替代方案帖**
```
搜索词示例：
- "[城市名] 工商档案 网上查询"
- "[城市名] 企业档案 线上调取"
- "[城市名] 电子档案 下载"
```

提取信息：
- 是否有线上查询渠道（如"粤商通"APP、"电子营业执照"小程序）
- 线上可查范围和限制
- 线上 vs 线下的内容差异

#### 2.2 联网搜索（官方信息）

**搜索目标：**

| 搜索内容 | 搜索词 | 目标 |
|---------|--------|------|
| 市场监管局官网 | "[城市名] 市场监督管理局 官网" | 获取官方公告的地址、电话、时间 |
| 档案查询公告 | "[城市名] 企业档案查询 办事指南" | 官方发布的查档流程和要求 |
| 政务大厅地址 | "[城市名] 政务服务中心 市场监督管理局窗口" | 确认窗口是否在政务大厅 |
| 预约平台 | "[城市名] 市场监管局 档案查询 预约" | 找到在线预约入口 |
| 百度/高德地图 | "[单位名称] 地址 电话" | 地图上的位置、电话、评论 |

**搜索结果优先级：**

1. 政府官网公告（`gov.cn`） → 最准确但可能过时
2. 12345 政务服务热线问答 → 较准确
3. 律协/律所公众号文章 → 律师视角的实操经验
4. 小红书笔记 → 最新鲜的个人体验
5. 百度知道/知乎 → 参考参考

#### 2.3 交叉验证

将小红书经验帖和官方公告进行交叉比对：

| 信息项 | 官方公告 | 小红书经验 | 置信度 | 最终采纳 |
|--------|---------|-----------|--------|---------|
| 工作时间 | 9:00-17:30 | "实际上11:30就停止取号了" | ⚠️ 实践为准 | 建议上午去 |
| 所需材料 | 律师证+介绍信 | "还要带案件受理通知书" | ✅ 以多的为准 | 都带 |
| 地址 | XX路XX号 | "搬到政务大厅3楼了" | 🔴 查证后采纳 | 以最新为准 |

> **交叉验证规则**：
> - 多人提同一信息 → 可信度高
> - 小红书近3个月的帖 → 时效性好
> - 官方公告有明确更新时间 → 优先
> - 信息矛盾 → 标注矛盾，建议电话确认

--- 输出阶段提示 ---
完成搜索后，输出搜索摘要：
```
🔍 查档信息搜索完成：
- 小红书：找到 X 篇相关经验帖（最近一篇：YYYY-MM-DD）
- 官网公告：X 份办事指南
- 地图信息：X 个相关地点
- ⚠️ 信息冲突点：X 处（已在指引中标注）
```

## 四、现场流程

```
到达政务中心
    ↓
凭律师执业证取号（选择"档案查询" / "企业档案"）
    ↓
等待叫号（根据小红书经验：一般等待 10~30 分钟）
    ↓
到窗口出示材料
    ↓
填写《企业档案查询申请表》（如需）
    ↓
工作人员调取电子档案
    ↓
选择：打印纸质（盖章）/ 刻录光盘 / 拍照留存
    ↓
缴费（如有打印费用）
    ↓
领取档案材料 + 收据
```

> ⏱️ 预计全程耗时：30~60 分钟（如档案量大可能更长）

## 五、注意事项

### ⚠️ 时间相关
1. **建议上午去**，越早越好——下午部分窗口不办理查档业务
2. 部分窗口周三下午/周五下午固定不对外（出发前电话确认）
3. **法定节假日前最后一个工作日**——窗口可能提前下班

### ⚠️ 材料相关
4. 介绍信抬头格式务必准确（窗口可能因抬头错一个字就拒办）:
   - 错误示例："深圳市工商局"
   - 正确示例："深圳市市场监督管理局"
5. 授权委托书上的被查企业名称必须与工商登记名称**完全一致**（含"有限公司"后缀）
6. 介绍信有效期一般7-15天，**开好后3天内去**最稳妥
7. 如查档人为非本所律师，需提供该律师的执业证复印件+其所在律所的证明函

### ⚠️ 现场相关
8. 部分窗口需要现场填写《企业档案查询申请表》，**不会填就问窗口人员指导**
9. 如被拒绝查档，礼貌询问原因和法律依据，记下工号，可打12345投诉
10. 查完档案先别走——当场翻阅，确认所需页面都已打印/拍照

### 🎯 执行重点
11. **必查页面**：验资报告（实缴真实性）、股权转让协议（低价/无偿转让线索）、减资决议（是否通知债权人）
12. 如发现**判决后突击转让股权**，立即标注——可作为撤销权之诉线索
13. 如发现**注册资本认缴未实缴**，固定证据——追加被执行人依据

### 📱 实用工具
14. 提前下载"电子营业执照"小程序/当地政务 APP（如粤商通、浙里办、随申办）——部分地区支持线上查档
15. 微信搜索"[城市名] 企业档案查询"关注当地市场监管局公众号的最新通知

## 特殊场景处理

### 场景一：搜索不到该城市的查档经验

小红书和搜索均未找到该城市专门的经验帖 → 使用**通用指引**：

1. 推断查档单位（一般规律）：
   - 公司 → 市级市场监督管理局
   - 个体户 → 区级市场监督管理局
2. 给出通用材料清单（律师执业证+介绍信+授权委托书）
3. 标注"⚠️ 未找到该城市的具体查档经验，以上为通用指引，建议出发前拨打 12345 确认"
4. 建议用户在指引中补充实地体验

### 场景二：信息已被证实过时

1. 如官网公告日期超过2年 → 标注"⚠️ 公告发布于 20XX 年，信息可能已变更"
2. 如多家小红书笔记提到窗口已搬迁 → 以最新地址为准
3. 无法判断哪条信息准确 → 标注矛盾 + 建议电话确认

### 场景三：城市级别调整

如注册地为县级市或县城：
1. 先查县级市场监管局
2. 如县级不支持，引导去市级
3. 标注"建议先电话确认县级市场监管局是否保存内档（部分县级单位内档已移交市级统一管理）"

### 场景四：疫情期间的特殊安排

1. 搜索是否有"不见面审批""线上预约"等安排
2. 如当地已全面线上化，标注线上流程替代线下指引

## 执行约束

1. **时效性**：指引中的窗口信息有时效性，出发前务必电话确认
2. **责任边界**：本指引为信息收集，不能替代电话确认。如因窗口变动导致白跑，本指引不承担责任
3. **信息标注**：所有信息必须标注来源和时间，用户可自行判断可信度
4. **不编造**：搜索不到的信息标注"未找到相关信息"，不推测不编造
